About

      
          - Home assistant and IoT system utilizing a Raspberry Pi and Coral Tensor Processing Unit.

          - Consists of a web application and Telegram bot employing AI/ML integration with a fault tolerant design.

          - Exhibits a quantized multi-object machine learning classifier for realtime video object-detection.


          - Follows a multi-threaded design to reduce process-blocking bottlenecks.

 
          - Employs a redundant (SMS/Telegram) notification system for quick alerts of activity.


          - Includes reminder system with recurrence optioning.


          - Seamlessly integrates a home automation system which is facilitated by the system's Mosquitto Broker via the MQTT protocol. This enables low bandwith/latency bidirectional communication.


          
          - Users can connect IoT devices and sensors via Wifi and Rest-API technologies.


          - Sensors asyncronously push updates only when needed to conserve bandwidth and/or battery.


          - Home automation comprises of scenes and scene actions to automate home tasks like turning devices on/off and temperature control.


          - Bot provides telegram services such as; server stats, device interfacing, jokes, facts, and trivia.


          - Bespoke job engine asynchronously schedules tasks and services as background processes for reminders, scenes, etc.


          - Personal data is secured by an AES algorithm in CBC (Cipher Block Chaining) mode with a 128-bit key for encryption which uses PKCS7 padding.
          


          
        
        
      

      

      

      
      
Hardware Specifications

      - Processor: Broadcom BCM2711, quad-core Cortex-A72 (ARM v8) 64-bit SoC @ 1.5GHz

      - Linux Kernel: Linux raspberrypi 6.1.21-v8+

      - Connectivity: 2.4 GHz and wireless LAN, Bluetooth, Gigabit Ethernet
      - Multimedia: H.265 H.264 OpenGL ES, 3.0 graphics
      - Flash Memory: SanDisk 64GB Extreme microSDXC UHS-I Micro SD Card
      - Input power: 3A 5V DC (USB-C or GPIO header) or PoE (requires PoE HAT)

      Google Edge Tensor Processing Unit: Used for high-speed ML inferencing.
      - Supported OS: Debian based Linux
      - Model: G950-06809-01
      - IO: up to 4 tera-ops per second (tops)
