Install Instructions

        
Mobile


        - Navigate to the home page
        - Click the three dots on top-right 
          of browser
        - Select "Add to Home screen"
    
        
Desktop


        - Paste the following url in the browser:

            chrome://flags/

        - Search for insecure origins
        - Enter home url into the input box
        - Enable the setting and restart browser
        - Install button will now be enabled
