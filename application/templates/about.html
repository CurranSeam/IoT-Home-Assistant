About





Services


IoT (Internet of Things)

- Home automation system ran via a web or standalone app, and Telegram.

- Comprises of scenes and scene actions to automate home tasks like turning devices on/off and temperature control.

- Users can connect IoT devices and sensors via Wifi and Rest-API technologies.

- Employs a redundant (SMS/Telegram) notification system for quick alerts of activity.

Scheduling

- Set reminders and have the Telegram bot notify you.

- Telegram bot sends all active users a morning message with a joke.

- The system schedules and runs scenes for users.

Telegram

- In the Telegram app, the Bot provides services such as; server info, device interfacing, jokes, facts, and trivia.




Architecture


- System is ran via a Raspberry Pi and Coral Tensor Processing Unit.

- The system's MQTT Mosquitto Broker enables low bandwith/latency bidirectional communication.

- Applications employs AI/ML integration with a fault tolerant design.

- Exhibits a quantized multi-object machine learning classifier for realtime video object-detection.

- Follows a multi-threaded design to reduce process-blocking bottlenecks.

- Includes reminder system with recurrence optioning.

- Sensors asyncronously push updates only when needed to conserve bandwidth and/or battery.

- Bespoke job engine asynchronously schedules tasks and services as background processes for reminders, scenes, etc.

- Personal data is secured by an AES algorithm in CBC (Cipher Block Chaining) mode with a 128-bit key for encryption which uses PKCS7 padding.
